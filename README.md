## 主题安装

### 运行环境
**运行环境**：Nginx，其他环境可自行尝试
**PHP版本**：PHP 7.4+，强烈推荐 PHP 8.3 或 PHP 8.4，强烈建议安装 opcache 扩展
**Typecho版本**：1.3.0+

### 主题安装
1. 将 Mirai 文件夹上传至 Typecho 安装目录下的 usr/themes/ 文件夹中

2. 将 MiraiCore 文件夹上传至 Typecho 安装目录下的 usr/plugins/ 文件夹中

3. 登录 Typecho 后台，进入「控制台」→「插件」

4. 在插件列表中找到「MiraiCore」，点击「启用」

5. 进入「控制台」→「外观」

6. 在主题列表中找到「Mirai」，点击「启用」

7. 点击主题名称进入「设置外观」进行配置

### 伪静态配置
必须设置伪静态规则，且在后台启用地址重写

Nginx伪静态规则
```bash
location / {
        try_files $uri $uri/ /index.php$is_args$args;
    }
```
保存好后，重启Nginx环境，然后再到后台「永久链接」保存一次即可


## 主题核心优势

### 优势一：极致 SEO 架构
Mirai 主题严格遵循搜索引擎排名规则，内置了全套 SEO 增强方案，让您的网站在竞争中脱颖而出。

- **全自动 Schema 结构化数据**： 自动生成符合 Schema.org 标准的 JSON-LD 标记（包括 WebSite、Article、Breadcrumb 等），显著提升搜索结果页（SERP）的点击率与专业展现。

- **智能 Meta 标签优化**： 针对百度与谷歌算法，自动优化 Title、Description 与 Keywords。同时支持 Open Graph 与 Twitter Card，让社交分享自带流量。

- **即时索引推送**： 集成百度主动推送与 IndexNow（必应等），确保新发布的文章在几秒钟内被搜索引擎抓取。

- **规范化链接处理**： 自动添加 Canonical 规范化标签与外链 nofollow 属性，防止权重流失。

### 优势二：惊叹的性能与加载速度
速度是用户体验的核心，也是 Google 排名（Core Web Vitals）的关键指标。

- **全场景图片优化**： 采用 lazysizes 智能懒加载技术，并原生支持 WebP/AVIF 现代格式，在不损失画质的前提下减少 50% 以上的图片体积。

- **极致预加载技术**： 支持 DNS Prefetch（预解析） 与 Instant Page（鼠标悬停预加载），实现物理意义上的“点击即打开”。

- **高效代码处理**： 关键 CSS 内联、非关键资源异步加载，配合 CDN 全球加速支持，打造极速访问体验。

### 优势三：全终端自适应响应式布局
完美契合移动端优先（Mobile-First Indexing）策略。

- **动态网格系统**： 桌面端支持 1-4 列布局自由切换，移动端精细化优化。

- **动态模式切换**： 支持跟随系统自动切换或用户手动偏好设置，平滑的过渡动画让阅读更具仪式感。

## 主题的功能亮点

### 功能一：付费阅读与变现系统
Mirai 是目前 Typecho 圈内最完善的商业化主题之一。

- **灵活付费模式**： 支持“整篇付费”或“部分隐藏付费” ，适配多种内容分发场景。

- **多渠道支付接入**： 内置易支付、支付宝当面付、微信、QQ 支付及余额支付，覆盖主流支付习惯。

- **创作者激励**： 完善的作者分成系统、实时收益统计与提现功能，非常适合搭建多作者知识付费平台。

- **完整的余额系统**： 用户可充值、消费、查询订单，构建闭环的虚拟经济系统。


### 功能二：沉浸式文章与创作体验
基于 Markdown 的深度定制，让写作与阅读都成为享受。

- **增强型编辑器**： 集成 Editor.md，支持实时预览、代码高亮（highlight.js）及深浅色适配。

- **交互式功能**： 提供文章点赞、收藏、阅读时间预估、自动生成目录（TOC）以及图片灯箱功能。

- **人性化设计**： 时间显示支持“刚刚、X分钟前”等相对时间，增强社区互动感。

### 功能三：用户中心与社交互动
打破 Typecho 传统的“单机”属性，建立活跃的社区生态。

- **前端会员系统**： 支持前端登录/注册、邮箱验证码验证、个人资料管理及自定义头像/封面。

- **评论系统增强**： 自动显示评论者 IP 归属地，支持嵌套评论、邮件提醒及多种 IP 查询接口。

- **导航与友情链接**： 支持多级菜单、自定义图标，以及专门的友情链接分类管理页面。
