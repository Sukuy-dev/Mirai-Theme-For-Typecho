## Mirai 主题下载与安装

### 运行环境要求

- **服务器环境**：Nginx（推荐），其他环境可自行尝试
- **PHP 版本**：PHP 7.4+，强烈推荐 PHP 8.3 或 8.4，建议安装 opcache 扩展
- **Typecho 版本**：1.3.0+
- **依赖插件**：MiraiCore 核心插件（必须安装并启用，否则主题无法运行）

### 安装步骤

1. 将 `Mirai` 文件夹上传至 Typecho 目录下的 `usr/themes/`
2. 将 `MiraiCore` 文件夹上传至 Typecho 目录下的 `usr/plugins/`
3. 登录后台 → 控制台 → 插件 → 启用「MiraiCore」
4. 进入控制台 → 外观 → 启用「Mirai」主题
5. 点击主题名称进入「设置外观」完成配置

### Nginx 伪静态配置

后台需启用地址重写功能，Nginx 伪静态规则如下：

```nginx
    if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php$1 last;
    }
```

保存后重启 Nginx，再到后台「永久链接」保存一次即可生效。

## Mirai 主题 SEO 架构优势

搜索引擎优化（SEO）是 Mirai 主题的核心设计目标之一。内置全套增强方案，覆盖结构化数据、Meta 标签、索引推送、链接规范化四大维度，帮助网站在百度、谷歌、必应等搜索引擎中获得更好的收录与排名。

### 全自动 Schema 结构化数据

自动生成符合 Schema.org 标准的 JSON-LD 标记，共支持 **6 种 Schema 类型**：

- **WebSite / Organization**：首页站点信息结构化
- **Article**：文章页富摘要展示（标题/作者/发布时间/封面图）
- **BreadcrumbList**：面包屑导航层级标记
- **CollectionPage**：分类页与标签归档页
- **SearchResults**：搜索结果页

覆盖全部页面类型，显著提升搜索引擎结果页（SERP）的点击率与专业展现形式。

### 智能 Meta 标签优化

针对不同搜索引擎算法自动优化 Title、Description 与 Keywords 标签，同时内置以下能力：

- **Open Graph 协议**：社交平台分享时自动提取标题、描述、封面图
- **Twitter Card**：Twitter/X 平台分享卡片优化
- **文章级 SEO 字段**：每篇文章可独立设置 SEO 关键词、SEO 描述、自定义封面图
- **Canonical 规范化**：自动输出规范链接，防止重复内容导致权重分散
- **外链 nofollow**：站外链接自动添加 nofollow 属性，保护网站权重不流失

同步移除 pingback 与 xmlrpc/wlw 等冗余请求，减少无效爬虫干扰。

### 即时索引推送

集成两大即时索引通道，确保新发布内容在几秒内被搜索引擎抓取：

- **百度主动推送**：文章发布后自动推送到百度站长平台
- **IndexNow 协议**：一键推送至必应（Bing）、Yandex、Seznam 等支持该协议的搜索引擎

此外还提供 RSS 2.0 与 Atom 双格式订阅输出，付费阅读文章自动过滤隐藏内容，保障订阅用户体验。

## 性能优化与响应式设计

页面加载速度是用户体验的核心指标，也是 Google Core Web Vitals 排名算法的关键因子。Mirai 从图片处理、资源加载、代码执行三个层面进行全方位性能优化。

### 图片懒加载与现代格式

采用 lazysizes 智能懒加载技术，配合原生 `<picture>` 响应式标签与 `srcset` 自适应属性，原生支持 WebP 与 AVIF 现代图片格式，在不损失画质的前提下大幅减少传输体积。**代码高亮组件仅在文章包含代码块时按需加载**，避免无关页面引入冗余资源。

### 预加载与异步执行

- **DNS Prefetch 预解析**：预解析外部域名，减少 DNS 查询延迟
- **Instant Page 预加载**：鼠标悬停链接时预加载目标页面，实现"点击即打开"
- **脚本异步加载**：所有 JavaScript 采用异步方式加载，非关键资源延迟执行
- **关键 CSS 内联**：首屏渲染关键样式内联到 HTML，其余按需加载

引入 RemixIcon 图标字体替代内联 SVG，图标统一管理更轻量。内置多级缓存机制减少重复计算开销。

### 移动端优先的响应式布局

完美契合 Google Mobile-First Indexing 策略，全终端自适应：

- **动态网格布局**：桌面端支持 1/2/3/4 列自由切换，移动端精细化适配
- **暗色模式**：跟随系统自动切换或用户手动偏好，浅色/深色模式可分别设置独立 Logo
- **移动端专属交互**：独立侧滑菜单（与桌面端完全分离）、底部固定 Tab 导航栏
- **悬浮导航**：导航菜单采用现代悬浮卡片样式，Apple Touch Icon 支持 iOS 主屏幕添加

## 付费阅读与 VIP 会员体系

提供从内容定价到资金流转的完整闭环，是目前 Typecho 生态中商业化程度最高的主题方案之一。

### 灵活的付费阅读模式

支持两种付费模式以适配不同的内容分发场景：

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 整篇付费 | 全文锁定，购买后完整阅读 | 付费课程、深度报告 |
| 部分隐藏付费 | 用 `[pay]...[/pay]` 标签包裹需付费段落 | 小说连载、混合免费+付费 |

每篇文章可在后台独立设置价格、购买方式限制（在线支付/余额支付/仅在线/仅余额）、是否强制登录、作者分成比例等参数。

### 多渠道支付接入

| 支付方式 | 说明 |
|---------|------|
| 易支付网关 | 聚合微信/支付宝/QQ 支付 |
| 支付宝当面付 | 官方扫码支付接口 |
| 余额支付 | 用户充值余额后直接抵扣 |

订单支持超时自动关闭（超时时间后台可配置）。完整的余额系统允许用户充值、消费、查询订单记录，货币单位可自定义。

### 三级 VIP 会员等级系统

Mirai 内置完整的 **三级会员体系**（一级/二级/三级），是付费阅读之外的第二大变现引擎：

- **5 种时长套餐**：月度（30天）/ 季度（90天）/ 半年（180天）/ 年度（365天）/ 永久（不过期），每种时长独立定价
- **4 种折扣模式**：每篇文章可单独选择——沿用全局折扣比例 / 自定义各等级价格 / 会员免费阅读 / 不参与会员折扣
- **灵活升级机制**：低等级会员升级高等级时自动计算补差价；续费支持原价续费 / 续费打折 / 续费固定价 / 不允许续费四种模式
- **自定义命名**：三级会员名称均可后台自定义，默认为一级会员/二级会员/三级会员

创作者激励方面，支持全局默认分成比例与单篇文章单独覆盖，包含实时收益统计与提现申请功能（可配置最小提现额与手续费）。

## 文章阅读体验与用户中心

### 沉浸式文章体验

基于 Markdown 深度定制，从写作到阅读的全链路优化：

- **增强型编辑器**：集成 Editor.md（含 CodeMirror），支持实时预览、代码高亮及深浅色适配。前端写文章支持草稿保存、发布/审核状态流转。
- **文章扩展字段**：每篇文章可设置封面图片、浏览量、SEO 关键词、SEO 描述、文章摘要共 5 个字段，后台可视化配置
- **丰富交互功能**：点赞、收藏、打赏（微信/支付宝/QQ 二维码弹窗）、社交分享按钮组、自动目录生成（TOC）、图片灯箱（Lightbox）
- **相关文章推荐**：按标签或分类匹配推荐，支持数量配置与多种排序方式

### 前端用户中心

打破 Typecho 传统"单机"模式，建立活跃的用户社区。用户中心覆盖个人管理、内容创作、财务交易三大板块：

| 板块 | 包含页面 | 功能说明 |
|------|---------|---------|
| 个人管理 | 概览面板 / 个人资料 / 密码修改 | 信息总览、VIP 状态、头像封面定制 |
| 内容创作 | 我的文章 / 写文章 / 我的评论 | 前台 CRUD 管理、Markdown 编辑器草稿/发布/审核、评论历史 |
| 社交互动 | 我的收藏 / 我的点赞 | 收藏列表与点赞记录查看 |
| 财务交易 | 钱包充值 / 订单记录 / 收入明细 / VIP 会员 | 多方式充值、订单追踪、收益统计提现、会员购买续费升级 |

支持前端登录/注册、邮箱验证码验证，登录后导航栏自动显示"我的"入口直达用户中心。

## 评论系统、邮件通知与社区功能

### 增强型评论系统

自动显示评论者 IP 归属地（支持多种 IP 查询接口），支持嵌套评论与评论分页，可根据权限和时间智能显示不同的评论状态提示。

### SMTP 邮件通知体系

完整的邮件模板系统，覆盖新评论通知管理员、评论回复通知评论者、注册/重置验证码发送等场景，后台可视化配置发件人信息与 SMTP 参数。

### 导航、友链与其他模块

- **多级导航菜单**：支持纯分类/纯页面/混合三种来源模式，RemixIcon 自定义图标
- **友情链接页面**：左侧分类锚点导航、热门站点推荐卡片、推荐文章整合展示
- **首页精选推荐**：三栏卡片式布局（左大图+右双小图），支持置顶推荐与 CID 指定或自定义图片配置
- **边栏组件**：管理员信息卡（含背景图）、热门文章、最新文章、彩色标签云、最新评论共 5 种模块，位置与顺序可调
- **作者归档页**：专属作者主页，含封面、头像、简介及文章列表
- **全屏搜索与面包屑导航**
